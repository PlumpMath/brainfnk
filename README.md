# brainfuck

A Clojure Brainf*** interpreter written at West London Hack Night.

# Usage

```sh
lein run "<program code>"
```

...for example:

### Hello World
```sh
lein run "++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>."

>> Hello World!
```
